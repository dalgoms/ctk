## Project Overview

이 프로젝트는 CTK·ChoiceDX의 브랜드 성장 가능성을 검토하기 위해 제작된 전략 문서입니다.

주요 목적은 단순한 웹사이트 개선이 아니라, ChoiceDX가 검색엔진과 AI 검색 환경에서 더 잘 발견되고 이해될 수 있도록 브랜드 구조, 정보 구조, 콘텐츠 전략, 실행 로드맵을 정리하는 것입니다.

## Key Direction

ChoiceDX의 핵심 과제는 새로운 자산을 만드는 것보다, 이미 보유한 리테일 측정 자산과 기술력을 시장에서 발견 가능한 구조로 전환하는 것입니다.

이를 위해 본 프로젝트는 다음 흐름으로 구성되었습니다.

1. 경쟁사와 시장 구조를 먼저 분석한다.
2. ChoiceDX가 이미 보유한 자산과 노출 공백을 진단한다.
3. SEO/GEO 관점에서 발견 가능한 정보 구조를 설계한다.
4. 브랜드 메시지와 실행 로드맵으로 전환한다.
5. 웹 기반 문서로 정리해 공유와 검토가 가능하도록 만든다.
