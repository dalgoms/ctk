# CTK ChoiceDX Missing Information Checklist

## 기준

### 문제
- 현재 프로젝트는 운영 구조, 원칙, 실행 계획은 정리되어 있지만 실제 분석과 실행에 필요한 입력 정보가 아직 부족하다.

### 원인
- 브랜드 성장 프로젝트는 시장, 경쟁사, 제품, 웹사이트, SEO, SNS, 영업자료, UX, 법무, KPI 정보가 함께 연결되어야 한다.

### 해결
- 부족한 정보를 우선순위별로 수집하고, 수집된 정보는 Project Context, Guide, Execution Plan, Deliverables에 반영한다.

### 실행
- `P0`: 프로젝트 시작 전 반드시 필요
- `P1`: 전략 수립 전 필요
- `P2`: 실행 고도화 및 검수 단계에서 필요

### 성과
- 분석, 전략, 실행, 검수, 인수인계가 추측이 아니라 실제 자료 기반으로 진행된다.

## 우선 수집 순서

### 문제
- 모든 정보를 한 번에 수집하려고 하면 프로젝트 시작이 늦어진다.

### 원인
- 분석과 전략에 반드시 필요한 정보와 실행 고도화에 필요한 정보가 섞여 있다.

### 해결
- P0부터 수집하고, P1은 7월 1-2주차에 병행 수집하며, P2는 실행 단계에서 보완한다.

### 실행
1. 웹사이트 URL, 제품 정의, 타깃 고객, 직접 경쟁사, KPI, 법무 기준을 먼저 확보한다.
2. 경쟁사 URL, 현재 SEO 상태, 브랜드 메시지, Figma 링크, 기존 영업자료를 확보한다.
3. 시장 데이터, SNS 성과, PR 자료, 고객 질문, 기술 검증 자료를 보완한다.

### 성과
- Week 1 Diagnosis와 Week 2 Strategy가 자료 부족으로 지연되지 않는다.

## Immediate Questions

### 문제
- 바로 분석을 시작하려면 최소 입력값이 필요하다.

### 원인
- 현재 가장 중요한 P0 정보가 아직 확정되지 않았다.

### 해결
- 아래 질문에 먼저 답하면 진단 작업을 시작할 수 있다.

### 실행
1. CTK ChoiceDX 공식 웹사이트 URL은 무엇인가?
2. 직접 경쟁사 3-5곳은 어디인가?
3. CTK ChoiceDX의 핵심 제품 설명 한 문장은 무엇인가?
4. 가장 중요한 타깃 고객은 누구인가?
5. 이번 프로젝트의 1순위 KPI는 무엇인가?
6. 법무상 반드시 피해야 하는 표현은 무엇인가?
7. 현재 Sales Deck, 브로슈어, Figma 링크가 있는가?

### 성과
- 위 7개 정보가 확보되면 현황 분석, 경쟁사 분석, 검색노출 분석을 바로 시작할 수 있다.
